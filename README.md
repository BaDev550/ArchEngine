### Building the Engine

1. **Clone the Repository**:

   ```bash
   git clone https://github.com/BaDev550/ArchEngine
   cd ArchEngine
   ```

2. **Generate Project Files**:

   Run the provided batch file to generate Visual Studio project files:

   ```bash
   Gen-Project-vs22.bat
   ```

3. **Open the Solution**:

   Open `ArchEngine.sln` in Visual Studio.

4. **Build the Solution**:

   Build the solution in Visual Studio to compile the engine and sample projects.
